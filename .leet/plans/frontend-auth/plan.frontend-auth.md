# Frontend Google Auth

Implement frontend pieces for Google OAuth login flow:

- Create LoginPage.vue with Google Sign-In button
- Create composable useAuth.js for auth state and actions
- Create Pinia auth store module
- Add auth guard to frontend router
- Add unit tests for composable and component

Environment variables:
- VITE_API_BASE (e.g. http://localhost:8000)
- VITE_GOOGLE_CLIENT_ID

Bootstrap requirements satisfied:
- start.sh/stop.sh present at repo root
- README updated with quick start
